## About

GAMES-JS is a small collection of classic single-player and two-player games, built from the ground up in pure HTML, CSS, and JavaScript.

No Frameworks are used.


## Games
* Snake
* Space Invaders
* Pong
* Minesweeper
* Asteroids
* Chess
